# Customer Churn AI Dashboard

## Overview
This project package is a Customer Churn AI Dashboard starter for Power BI, SQL, and machine learning.

It demonstrates:
- churn probability scoring
- key driver analysis
- retention action recommendations
- Power BI executive reporting
- optional Python ML workflow

## Included Files
- data/FactCustomerChurn.csv
- data/DimDate.csv
- sql/schema.sql
- sql/views.sql
- python_ml/train_churn_model.py

## Recommended Dashboard Pages
1. Executive Churn Overview
2. Churn Driver Analysis
3. High-Risk Customer Monitor
4. Retention Action Center
5. Model / Probability Insights

## Core KPIs
- Customers
- Churned Customers
- Churn Rate %
- Avg Churn Probability
- High Risk Customers
- Revenue at Risk
